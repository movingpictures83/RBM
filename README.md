# RBM
# Language: R
# Input: CSV (training set)
# Output: CSV (weights)
# Tested with: PluMA 1.1, R 4.0.0
# Dependency: RBM 

PluMA plugin that runs a Restricted Boltzmann Machine (Smolensky, 1986),
using the library of (Matzen, 2019): https://github.com/TimoMatzen/RBM

The provided example training data corresponds to the MNIST dataset (LeCun
and Cortes, 2010), also used by (Matzen, 2019) in their tutorial.
